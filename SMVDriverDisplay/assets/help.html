Run

You can run this page by veiwing the display screen, which you can navigate to by using either the home screen or the menu.

The logger can be started and create new logs using the buttons on the botton right of the display screen.

 

Configure

The configure screen offers you various options to customize the operation of this app such as:

  1. Arduino Address - Which is the bluetooth address of the arduino that is running the data collection app.
  2. Arduino UUID - Which is the UUID code of the arduino that is running the data collection app.
  3. Refresh  Rate - Which is the time, in millisecounds, the app will refresh the screen.
  4. Log Rate - Which is the time, in millisecounds, the app will log the data.
  5. Log Directory - Which is the directory the logs will be place on your device.
