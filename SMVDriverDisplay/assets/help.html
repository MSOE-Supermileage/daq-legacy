Run

To run the dash board you must visit the display screen, which you can navigate to from either the home screen or the menu.

 

Data Logs

·         Start the logger by pressing the toggle button on the bottom right corner of the dash board

·         Create a new log file by clicking the “New Log” button on the bottom right corner

·         You can view the log files by navigating to the directory defined in the config screen

 

 

Configure

The configure screen offers you various options to customize the operation of this app such as:

1. Arduino Address - Which is the Bluetooth address of the arduino that is running the data collection app.

2. Arduino UUID - Which is the UUID code of the arduino that is running the data collection app.

3. Refresh Rate - Which is the time, in milliseconds, the app will refresh the screen.

4. Log Rate - Which is the time, in milliseconds, the app will log the data.

5. Log Directory - Which is the directory the logs will be place on your device.
